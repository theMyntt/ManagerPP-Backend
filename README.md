## Pre-requisites

- NodeJS v20.14.0
- Yarn v1.22.22
- MySQL Database

## Development use

- Clone this repo
- Install the dependencies with `yarn install`
- Configure your enviroments variable in `.env`
- Start project with `yarn start`

## Documentation

- The swagger will be at `/swagger-ui` route
